## Project goal
Build a premium author website for Alphonso Soosay using the existing Next.js starter scaffold.

## Stack
- Next.js
- React
- Tailwind CSS

## Requirements
- Preserve the premium editorial look and feel
- Convert navigation into real routes
- Refactor the starter into reusable components
- Use verified book metadata and supplied placeholders only
- Keep code maintainable and clean

## Pages
- `/`
- `/about`
- `/books`
- `/media`
- `/contact`

## Content rules
- Keep placeholder copy tasteful
- Do not invent specific factual claims unless marked as placeholder
- Keep final content easy to replace
- Avoid direct personal contact details and unconfirmed biography claims

## UX rules
- Responsive
- Accessible
- Minimal, tasteful motion
- Warm neutral luxury aesthetic

## Done means
- Project runs locally
- Clean structure
- No obvious build issues
- Ready for content replacement and deployment
